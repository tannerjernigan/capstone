==CSV file header and its meaning==
Location	Beach name
Profile		Transect names
Date		Survey date
Time_UTC	Survey time
Method		Either topography or bathymetry
X			Easting (ft), NC State Plane, NAD83
Y			Northing (ft), NC State Plane, NAD83
Z_NAVD88	Elevation (ft), NAVD88
DBL			Distance to baseline, i.e., the distance between surveyed location and an alongshore baseline
